Toda máquina segue esta estrutura:

1. **Reconhecimento** (Nmap, WhatWeb, Gobuster, FFUF)

3. **Exploração / Foothold** (Ganho de acesso inicial)
4. **Privilege Escalation** (User → Root/System)
5. **Post-Exploitation** (Flags, evidências)
